Global Configuration

API Settings

API settings used by the Alpaca API to retrieve live stock prices.

API Keys

Enter your Alpaca API keys here. They will be stored in your browser's local storage and will not be sent to our server.

API Cache Settings

Control how API responses are cached in your browser to reduce API calls and speed up the tools.

Cache Buster

When enabled, a version querystring (e.g., ?v=1.0.0) is added to CSS and JS files to prevent browsers from using old, cached versions during development.

Set the version string to apply.

Debug Settings

Control application-wide debugging and console logging.

Debug Mode

When enabled, diagnostic messages will be printed to the browser's console. This should be disabled in a production environment. The setting is saved in your browser's local storage.
